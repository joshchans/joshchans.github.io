Cycle Checking

Is there a cycle in the graph?
What is a cycle
Algorithms
Perform a DFS, starting from any vertex
Keep track of previous vertex during DFS
During the DFS, if the current vertex has an edge to an already-visited vertex which is not the previous vertex, then there is a cycle
After the DFS, if any vertex has not yet been visited, perform another DFS, this time starting from that vertex
Repeat until all vertices have been visited

Connected Components

üí°
How many connected components are there? Are two verticies in the same connect component?
What is a connected component
A connected component is a maximally connected subgraph
Algorithm
Goal: to compute an array which indicated which connected component each vertex is in.
Example:
Choose a vertex and perform a DSF starting at that vertex
During the DSF, assign all vertices visited to component 0
After the DSF, if any vertex has not been assigned a component, perform a DFS starting at that vertex
During this DSF, assign all vertices visited to component 1
Repeat until all vertices are assigned a component, increasing the component number each time.

How to check if a graph has an Eulerian path or circuit?
What is Eulerian path
Eulerian path: a path that visits each edge exactly once
Eulerian circuit: a path that starts and ends at the same vertex
Some helpful theorems
A graph has an Eulerian path if an only if exactly zero of two vertices had odd degree, and all vertices with non-zero degree belong to the same connected component.
A graph has an Eulerian circuit if an only if every vertex has even degree, and all vertices with non-zero degree belong to the same connected component.
